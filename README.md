
#  clients

## PROJECT DESCRIPTION

The goal is to have control over all clients belonging to the company OrionTek, where each client can have N number of addresses.

### Links, version jetpack compose
📌 Live site URL: [here](https://github.com/Tonyva002/clients)

### Links, version XML
📌 Live site URL: [here](https://github.com/Tonyva002/oriontek)

### Built with

- Kotlin
- Android
- coil
- Room
- Coroutines
- Hilt
- Flow
- JUnit 
- MVVM
- Clean Architecture
- Multilingual, English - Spanish
- Jetpack Compose

### Necessary tool:

Install Android Studio

### Steps to execute:

- Step 1. Download or Clone the repository
- Step 2. Unzip the file if you have downloaded it
- Step 3. Open it in Android Studio
- Step 4. Create an emulator or connect your physical phone to the pc.
- Step 5. Give it to run
